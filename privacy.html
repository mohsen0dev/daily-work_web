Privacy Policy — Work Days Logger

Last Updated: [Insert Date]

1. Information We Collect

Work Days Logger does not collect, store, or transmit any personal information. The only interaction with user data occurs when the app creates a local backup file or when the user chooses to upload this backup to their Google Drive.

2. Google Account Authentication

Google Sign-In is used solely to allow users to upload and restore their backup file in Google Drive. We do not access contacts, personal files, or any other content in the user's Google Drive except the specific backup file created by the application. We also do not store Google account data.

3. Data Storage

All work logs and application data are stored locally on the user’s device using the Hive database. No data is transmitted to external servers.

4. Backup on Google Drive

If the Google Drive backup feature is enabled, the app creates a backup file and stores it in a private folder within the user’s Drive. No one, including the developer, has access to this file. Users may delete the file at any time.

5. Third-Party Sharing

We do not share, sell, or transmit any data to third parties.

6. Security

All backup operations are performed securely through Google’s official API.

7. Children’s Privacy

The app does not target children and does not collect any personal information.
